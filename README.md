# Depression Detection from Reddit Posts using NLP and Machine Learning

This project focuses on **detecting signs of depression from Reddit posts** using **Natural Language Processing (NLP)** and **machine learning classification techniques**. The system analyzes both **textual content** (post titles and body text) and **user engagement metadata** (such as upvotes, comments, sentiment, and posting patterns) to classify whether a post indicates depressive tendencies.

The project follows a complete **end-to-end machine learning pipeline**, including:

* Data loading and exploratory data analysis (EDA)
* Text cleaning and preprocessing
* Tokenization, stopword removal, and lemmatization
* TF-IDF vectorization for text feature extraction
* Sentiment and statistical feature engineering
* Metadata transformation and feature scaling
* Hybrid sparse feature matrix creation
* Dataset balancing for efficient training
* Training multiple classification models
* Performance evaluation using accuracy, precision, recall, F1-score, confusion matrix, and ROC-AUC

Several machine learning models were trained and compared, including:

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Naive Bayes

The goal of this project is to explore how **social media language patterns and engagement signals** can be used to identify potential mental health concerns, contributing toward **early detection systems and AI-driven mental health support tools**.
